# Manage your business with Odoo

Odoo is an open-source business management suite that covers CRM, accounting, inventory, project management, website building, and more. Running Odoo on Olares gives you a fully self-hosted ERP system where all your business data stays under your control.

## Learning objectives

In this guide, you will learn how to:
- Install Odoo from Market and log in
- Change the interface language
- Build a website using the Website module

## Install Odoo

1. Open Market and search for "Odoo".

   <!-- ![Odoo in Market](/images/manual/use-cases/odoo.png#bordered) -->

2. Click **Get**, then **Install**, and wait for installation to complete.

3. Open Odoo from Launchpad, and log in with the default credentials:

   - **Username**: `admin`
   - **Password**: `admin`

   <!-- ![Odoo login page](/images/manual/use-cases/odoo-login.png#bordered) -->

## Change the interface language

Odoo supports multiple languages. To switch the interface language:

1. Click your username in the top-right corner and select **My Profile** (or **Preferences**).

   <!-- ![Odoo language settings](/images/manual/use-cases/odoo-language-settings.png#bordered) -->

2. In the **Language** field, select your preferred language.

   <!-- ![Select language](/images/manual/use-cases/odoo-select-language.png#bordered) -->

3. Click **Save**. The interface refreshes with the new language applied.

   <!-- ![Language applied](/images/manual/use-cases/odoo-language-applied.png#bordered) -->

## Build a website

Odoo includes a built-in Website module that lets you create and manage a website with a drag-and-drop editor. This section walks through enabling the module and building a basic site.

### Enable the Website module

1. From the Odoo home screen, find the **Website** module and click **Install**.

   <!-- ![Find Website module](/images/manual/use-cases/odoo-website-module.png#bordered) -->

2. Wait for installation to complete. Odoo opens the website builder automatically.

   <!-- ![Website builder](/images/manual/use-cases/odoo-website-builder.png#bordered) -->

### Create your website

1. Choose a template or start with a blank page in the website builder.

   <!-- ![Choose template](/images/manual/use-cases/odoo-website-template.png#bordered) -->

2. Use the drag-and-drop editor to add and arrange content blocks such as text, images, and buttons.

   <!-- ![Website editor](/images/manual/use-cases/odoo-website-editor.png#bordered) -->

3. Customize the design by adjusting colors, fonts, and layout settings.

   <!-- ![Customize website](/images/manual/use-cases/odoo-website-customize.png#bordered) -->

4. Click **Save** when you are done. Your website is now live and accessible.

   <!-- ![Completed website](/images/manual/use-cases/odoo-website-complete.png#bordered) -->

:::tip Explore more modules
The Website module is just one of many Odoo apps. From the home screen, you can install additional modules like CRM, Accounting, Inventory, and Project to extend your business operations.
:::

# Get help and support

You can get support for Olares through our community channels, browse self-help resources, or report issues directly to the development team.

## Self-help resources

Before asking the community, check the documentation and frequently asked questions to see if your issue has already been addressed.

* **[FAQs:](faqs.md)** Find answers to common questions about hardware, installation, and Olares features.
* **[Troubleshooting guide:](troubleshooting.md)** Follow step-by-step guides to diagnose and resolve common system errors.

## Community forums

Connect with other users to exchange ideas, share use cases, or get help with configuration.

* **[Olares forum:](https://forum.olares.com)** Best for long-form discussions, checking the roadmap, and sharing public use cases.
* **[Discord:](https://discord.gg/olares)** Best for real-time chat and quick exchanges with other community members.
* **[GitHub discussions:](https://github.com/beclab/olares/discussions)** Best for asking general questions and sharing feedback on specific repositories.

## Report issues

If you encounter a bug or technical error, you can submit a report directly to the developers on [GitHub issues](https://github.com/beclab/olares/issues).

Include the following information if possible:
- Steps to reproduce the issue.
- Any error messages or unexpected behaviors.
- Details about your environment (e.g., operating system, Olares version).

It's recommended to attach the complete system logs. For detailed instructions, see [Export system logs](../olares/settings/developer.md#export-system-logs).

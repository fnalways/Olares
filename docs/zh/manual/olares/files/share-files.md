# 分享文件

本功能支持将文件夹通过不同方式分享给内部成员或外部人员。你可以灵活设置访问权限、密码及有效期，并随时管理或撤销这些分享。

## 分享类型与支持范围

目前文件分享仅支持**文件夹**级别。根据文件夹所在的目录位置，支持的分享类型有所不同，请参照下表：

| 目录类型 | 公开分享 | 内部分享 | SMB分享 |
| :--- | :---: | :---: | :---: |
| **首页** | 支持 | 支持 | 支持 |
| **外部设备** | - | 支持 | 支持 |
| **同步** | - | 支持 | - |
| **数据** | 支持 | 支持 | 支持 |
| **缓存** | - | 支持 | 支持 |

其中：
- **公开分享**：生成访问链接，适用于对外分享（支持密码保护）。
- **内部分享**：仅限同一 Olares 集群内的成员访问，适用于内部团队协作。更多信息请参阅 [管理多用户](../settings/manage-team.md)。
- **SMB分享**：通过SMB协议在局域网内共享。

![Share files type](/images/manual/olares/share-files-1.png#bordered)

## 用户权限类型

你可以为**内部分享**和**SMB 分享**的成员分配特定权限，以控制他们的访问级别和允许的操作。

| 权限 | 说明 |
| :--- | :--- |
| **管理员** | 用户拥有完全控制权，包括：添加、查看、编辑和删除文件；<br>邀请和移除成员；修改分享设置。 |
| **编辑** | 用户可以添加、查看、修改和删除文件。 |
| **查看** | 用户仅能打开、查看和下载文件，无法上传、修改或删除任何内容。 |

## 创建分享

### 创建内部分享

1. 打开**文件管理器**。
2. 右键点击目标文件夹，然后点击**内部分享**。
3. 在**邀请用户**中，点击 <i class="material-symbols-outlined">add</i>，搜索并勾选目标用户或组，点击**邀请**。
  ![内部分享配置中邀请用户](/images/zh/manual/olares/internal-share-invite-1.png#bordered){width=60%}
4. 在**用户权限设置**列表中，点击用户头像右侧的 <i class="material-symbols-outlined">chevron_forward</i>，为每一位邀请用户分配具体的读写权限，设置完成后，点击**提交**。
5. 点击 **确认**。

### 创建 SMB 分享

SMB 分享使用一套专用的本地 Linux 账户体系。在此体系内生成的 SMB 账户独立于 Olares 成员，仅用于局域网文件访问。

1. 右键点击目标文件夹，然后点击**SMB 分享**。
2. 选择是否**公开**。
    * 如果选择**是**，点击**确认**，该文件夹将对局域网内的任何人开放访问。
    * 如果选择**否**，则需要身份验证。你必须通过**步骤 3** 或**步骤 4** 选择或创建一个专用的 SMB 账户。
3. 选择现有 SMB 账户进行授权。
    
    a. 在**邀请用户**栏中，点击 <i class="material-symbols-outlined">add</i>。
    
    ![SMB分享配置中选择已有SMB账户](/images/zh/manual/olares/smb-share-invite-1.png#bordered){width=60%}
    
    b. 搜索 SMB 账户，勾选该账户，然后点击**邀请**。
    
    c. 在**设置用户权限**列表中，点击用户头像右侧的 <i class="material-symbols-outlined">chevron_forward</i> 分配具体权限，然后点击**提交**。
    
    d. 点击**确认**。

4. 创建新的 SMB 账户进行授权。
    
    a. 点击**邀请用户**右侧的**添加用户账户**。

    ![SMB分享配置中创建新SMB账户](/images/zh/manual/olares/smb-share-new-usr-1.png#bordered){width=60%}
    
    b. 设置唯一的用户名。
    
    c. 在**设置密码**中，使用系统生成的安全密码。你也可以手动输入新密码，或点击**重新生成**来使用一个新的随机密码。
    
    d. 点击**确认**。系统将自动生成此账户。
    
    e. 在**邀请用户**栏中，点击 <i class="material-symbols-outlined">add</i>。
    
    f. 搜索新创建的账户，勾选该账户，然后点击**邀请**。
    
    g. 在**设置用户权限**列表中，点击用户头像右侧的 <i class="material-symbols-outlined">chevron_forward</i> 分配具体权限，然后点击**提交**。
    
    h. 点击**确认**。

### 创建公开分享

1. 打开**文件管理器**。
2. 右键点击目标文件夹，然后选择 **公开分享**。

![公开分享设置](/images/zh/manual/olares/public-share-settings-1.png#bordered){width=58%}

3. 在**设置密码**中，手动输入访问密码，或点击**重新生成**使用系统随机生成的安全密码。
4. 在**设置有效期**中，选择分享链接过期的具体时间。
5. 其他限制（按需选择）：
    * **限制文件大小**：勾选并填写数值，选择计量单位，即可限制访客上传文件的大小。
    * **仅允许上传**：勾选后，访问者仅能上传文件，不可查看或下载现有内容。
6. 点击**确认**生成链接。
7. 复制或记录下链接地址，点击**确认**。

## 查阅与管理分享

你可以在**分享**列表里集中查看和管理所有已分享的文件夹。

### 查阅分享列表及属性

1. 打开**文件管理器**。
2. 在左侧边栏中，点击**分享**。右侧将展示所有当前正在分享的文件夹列表。
3. 如果你想查看某个文件夹的详情，右键点击目标文件夹，然后点击**属性**。你将看到分享者、原始路径、分享范围、创建者、当前权限、过期时间及链接详情等。

### 修改密码或权限

对于已分享的文件夹，你可以随时更改其密码和访问权限。

1. 打开**文件管理器**。
2. 在左侧边栏中，点击 **分享**，右侧将展示**分享**列表。
3. 若需 **重置密码**（仅限**公开分享**）：
        
    a. 右键点击文件夹，选择**重置密码**。
        
    b. 输入新密码或点击**重新生成**。
        
    c. 点击**确认**。
4. 若需 **编辑权限**（仅限**内部分享**和**SMB分享**）：
        
    a. 右键点击文件夹，选择**编辑权限**。
        
    b. 添加或删除用户，或修改现有用户的权限。
        
    c. 点击**确认**。

### 撤销分享

如果你不再需要分享某个文件夹，可以随时撤销该分享。

1. 打开**文件管理器**。
2. 在左侧边栏中，点击**分享**，右侧将展示**分享**列表。
3. 右键点击目标文件夹，点击**撤销分享**。
4. 点击**确认**。

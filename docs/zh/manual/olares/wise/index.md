# 使用 Wise 一站式聚合、管理内容

Wise 是一款以本地优先和 AI 驱动的现代阅读器，是 Olares 生态系统中的核心应用之一。它帮助你从各种平台收集、阅读和管理信息，从而创建并维护个人信息中心。

Wise 内置两大核心能力：

- **跨平台内容聚合**：将不同来源的文章、视频、音频和订阅源统一整合为无干扰的阅读界面。
- **一键收藏**：使用 LarePass 浏览器扩展将网页直接保存到 Olares，确保你可以随时访问和管理内容。

## 安装并打开 Wise

1. 打开应用市场，搜索“Wise”。
2. 点击**获取**，接着点击**安装**。
3. 安装完成后，从启动台打开 Wise。

## 了解界面

了解 Wise 主界面，高效管理并玩转你的阅读空间。

![Wise 界面](/images/zh/manual/olares/wise-interface.png#bordered)

### 侧边栏导航

侧边栏导航分为以下部分：

- **订阅源**：这里是所有订阅内容的来源，包括网站、博客和视频频道。
- **收件箱**：这里是你处理当前内容的中心工作区，聚合了所有来源的项目，包括保存的网页、上传的文件（如 PDF 或 EPUB），以及添加到资料库的订阅内容。
- **内容类型**：Wise 会根据文件格式自动分类你的内容，如**文章**、**视频**、**书籍**。

### 文件状态

Wise 中的每个文件都只有一个状态，分别为**收件箱**或**稍后阅读**。你可以在列表视图或阅读视图中，点击图标更改其状态：

- **<i class="material-symbols-outlined">inbox</i> 收件箱**：表示该文件计划近期阅读。
- **<i class="material-symbols-outlined">schedule</i> 稍后阅读**：表示该文件计划稍后查看。

# 使用 Wise 一站式聚合、管理内容

Wise 是一款以本地优先和 AI 驱动的现代阅读器，是 Olares 生态系统中的核心应用之一。它帮助你从各种平台收集、阅读和管理信息，从而创建并维护个人信息中心。

Wise 内置两大核心能力：

- **跨平台内容聚合**：将不同来源的文章、视频、音频和订阅源统一整合为无干扰的阅读界面。
- **一键收藏**：使用 LarePass 浏览器扩展将网页直接保存到 Olares，确保你可以随时访问和管理内容。

## 了解界面

了解 Wise 主界面，高效管理并玩转你的阅读空间。

![Wise 界面](/images/zh/manual/olares/wise-interface.png#bordered)

### 侧边栏导航

侧边栏导航分为三个逻辑区域：

- **订阅源**

    这里是所有订阅内容的来源。你可以在此添加任何可订阅的内容（包括网站、博客和视频频道）。点击某个订阅源后，你可以在**未读**和**已读**之间切换其状态，跟踪阅读进度。

- **收件箱**

    这里是你处理当前内容的中心工作区，聚合了所有当前状态为**未读**的文件。这里的文件通常有以下三个来源：
    - 通过 LarePass 浏览器扩展或手动添加收藏的网页
    - 上传的文件，如 PDF、EPUB 等
    - 从订阅源保存到收件箱

- **文件类型**

    Wise 会根据文件格式自动对其分类（**文章**、**视频**、**音频**、**图书**、**PDF**）。每个类型文件夹中都包含两个根据文件状态划分的标签页：
    - **收件箱**：计划近期阅读的主要内容集合。
    - **稍后阅读**：计划稍后查看的内容。
    ![Item status](/images/zh/manual/olares/wise-tabs-in-folder.png#bordered)

### 文件状态

Wise 中的每个文件都只有一个状态，分别为**收件箱**或**稍后阅读**。你可以在列表视图或阅读视图中，点击图标更改其状态：

- **<i class="material-symbols-outlined">inbox</i> 收件箱**: 表示该文件计划近期阅读。
- **<i class="material-symbols-outlined">schedule</i> 稍后阅读**:表示该文件计划稍后查看。

更改文件状态将决定文件在侧边栏中的显示位置，如出现在**收件箱**，或对应文件类型的**收件箱**/**稍后阅读**标签页中，并随着状态切换，即时更新位置。
